# Programming-Hero-Assignment1

Question 1: What are some differences between interfaces and types in TypeScript?

TypeScript শিখতে গিয়ে আমি প্রথম যে জিনিসটা বুঝেছি, তা হলো interface আর type দেখতে একই রকম মনে হলেও, আসলে দুটোর কাজ করার পদ্ধতিতে কিছু গুরুত্বপূর্ণ পার্থক্য আছে। Interface মূলত কোনো object-এর গঠন নির্ধারণ করতে ব্যবহার হয় মানে একটি object-এর মধ্যে কী কী property থাকবে, কোনটার type কী হবে ইত্যাদি। Interface-এর একটা সুবিধা হলো, চাইলে আমি একই নামে আরেকটা interface লিখতে পারি। আর TypeScript এগুলোকে একসাথে merge করে দেয়। ফলে বড় project-এ ধীরে ধীরে structure বাড়ানোর সুবিধা থাকে।

Type আবার একটু অন্য রকম। এটা অনেক বেশি flexible। Object ছাড়াও union, primitive, tuple, function type—সবকিছু define করা যায়। তবে type-এর সীমাবদ্ধতাও আছে—আমি চাইলে একই নামে দুইবার type declare করতে পারি না, করলে immediately error দেয়। কিন্তু complex বা mixed type বানানোর ক্ষেত্রে type-ই সবচেয়ে ভালো কাজ করে।

সব মিলিয়ে, আমি এটা বুঝেছি যে—object-এর design বা model বানাতে interface ব্যবহার করা সহজ ও পরিষ্কার। আর যখন union বা বিভিন্ন ধরণের type মিশিয়ে কিছু তৈরি করতে হয়, তখন type বেশি শক্তিশালী এই পার্থক্যগুলো পরিষ্কারভাবে বুঝে গেলে TypeScript কোড লেখার সময় কোনটা কোথায় ব্যবহার করা উচিত, সেটা সিদ্ধান্ত নেওয়া অনেক সহজ হয়ে যায়।

Question 3:Explain the difference between any, unknown, and never types in TypeScript.

TypeScript-এ any, unknown, এবং never—এই তিনটি টাইপ মূলত তিন রকমের ভিন্ন পরিস্থিতির জন্য ব্যবহার করা হয়। any হলো সবচেয়ে ঢিলেঢালা টাইপ, এতে যেকোনো ধরনের মান রাখা যায় এবং চাইলে সেই মান দিয়ে যেকোনো কাজও করা যায় TypeScript কোনো ধরনের সতর্কতাই দেখায় না। তাই এটি সুবিধাজনক হলেও বড় প্রকল্পে অসতর্কভাবে ব্যবহার করলে অনেক ভুল ধরা পড়ে না। unknown দেখতে any-এর মতো মনে হলেও আসলে এটি বেশি নিরাপদ যদিও, এতে সব ধরনের মান রাখা যায়, কিন্তু সেই মান ব্যবহার করার আগে TypeScript বাধ্য করে টাইপ চেক করতে। ফলে কোড আরও নিরাপদ থাকে এবং ভুল কম হয়। আর never ঠিক বিপরীত এটি সেই জিনিসগুলো বোঝায় যা কখনোই ঘটবে না, যেমন এমন একটি ফাংশন যা সবসময় error ছোড়ে বা কখনো ফিরেই আসে না। সহজভাবে বলতে গেলে, any হলো পুরো স্বাধীনতা, unknown হলো স্বাধীনতা কিন্তু নিরাপত্তাসহ, আর never হলো এমন পরিস্থিতি যেখানে কোনো মান থাকার সুযোগই নেই।
